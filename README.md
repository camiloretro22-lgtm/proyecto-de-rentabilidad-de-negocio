<h1> 📊 Análisis de Rentabilidad de Negocio (Marketing + Producto) </h1>

<h1> 🧠 Descripción General </h1>

Este proyecto analiza el desempeño de un negocio digital a través de datos de visitas, pedidos y costos de marketing, con el objetivo de entender:

Cómo se comportan los usuarios desde la adquisición hasta la compra

Qué tan rentable es cada canal de marketing

Qué oportunidades existen para optimizar ingresos y reducir costos

El enfoque principal está en conectar datos → métricas → decisiones de negocio.

<h1> 🎯 Objetivo </h1>

Evaluar la rentabilidad del negocio mediante el análisis de:

Comportamiento de usuarios (engagement y retención)

Conversión a compra

Valor generado por cliente (LTV)

Costo de adquisición (CAC)

Retorno de inversión en marketing (ROMI)

👉 Todo esto con el fin de optimizar la asignación de presupuesto y maximizar ingresos.

<h1> 🛠️ Tecnologías Utilizadas </h1>

Python (análisis de datos)

Pandas / NumPy (manipulación de datos)

Matplotlib / Seaborn (visualización)

Jupyter Notebook (exploración y storytelling)

<h1> 📈 Métricas Clave Analizadas </h1>

👥 Producto (Usuarios)

DAU / WAU / MAU → nivel de actividad de usuarios

Duración de sesión → engagement

Sesiones por usuario → frecuencia de uso

💡 Insight:

El comportamiento de uso permite identificar si el producto genera hábito o uso puntual.

<h1> 💰 Ventas </h1>

Ingresos totales

AOV (Average Order Value) → ticket promedio

Tiempo hasta la compra

💡 Insight clave:

La mayoría de usuarios compran el mismo día (día 0), lo que indica un comportamiento de compra impulsiva o altamente influenciado por marketing.

<h1> 🔁 Cohortes y LTV </h1>

Análisis por cohortes mensuales

LTV (Lifetime Value) acumulado por usuario

💡 Insight:

Permite entender cuánto valor genera un cliente a lo largo del tiempo y detectar problemas de retención.

<h1> 📢 Marketing </h1>

CAC (Costo de Adquisición de Cliente)

ROMI (Return on Marketing Investment)

💡 Fórmula clave:

ROMI = LTV / CAC

<h1> 🔍 Hallazgos Clave (Impacto en Negocio) </h1>

1. 💸 Rentabilidad por canal (ROMI)

Se identificaron fuentes de tráfico con ROMI > 100% (rentables)

Otras fuentes generan pérdidas → oportunidad de optimización

📌 Impacto:

Redistribuir presupuesto hacia canales rentables puede aumentar significativamente el ROI global.

2. ⚡ Conversión inmediata

Alta proporción de usuarios compran el mismo día de la primera visita

📌 Impacto:

El marketing está atrayendo usuarios con alta intención

Se debe priorizar la inversión en adquisición más que en nurturing

3. 📱 Diferencias por dispositivo

Un dispositivo genera más ingresos que otros (ej. desktop vs mobile)

📌 Impacto:

Optimizar la experiencia en el dispositivo más rentable puede incrementar ingresos sin aumentar costos.

4. 📉 Retención limitada

El LTV muestra que el valor se concentra en los primeros períodos

📌 Impacto:

Existe oportunidad en estrategias de retención

Incrementar retención = mayor LTV sin aumentar CAC

<h1> 🧩 Conclusión </h1>

Este análisis demuestra cómo los datos pueden transformarse en decisiones estratégicas claras:

No se trata solo de medir métricas, sino de entender qué acciones generan rentabilidad real en el negocio.
